# Notiky

**The AI workspace where products are built** — from first idea to scale.

AI coding tools made you 10x faster at writing code. But 0x better at keeping your product coherent. Decisions live in your head, context scatters across ChatGPT threads, and every new AI session starts from scratch.

Notiky fixes this. One workspace: **brainstorm deeply, capture decisions, ship plans, generate artifacts** — and keep every tool in sync.

---

### What it does

- **AI co-founder brainstorming** — challenges your thinking, does market research, drills into specifics (not a yes-man chatbot)
- **Structured decisions** — who decided, when, why, what was rejected. Searchable forever
- **Ship plans** — ideas become tasks with full context, flowing to Linear/Jira
- **Artifact generation** — auto-generates `CLAUDE.md`, `.cursorrules`, design systems, API docs
- **Sync engine** — backend changes propagate to frontends. Nothing falls through the cracks
- **MCP server** — live product context delivered to Cursor and Claude Code mid-session

### Built with

`Next.js` `TypeScript` `Supabase` `Drizzle ORM` `Vercel AI SDK` `Tailwind CSS` `shadcn/ui`
