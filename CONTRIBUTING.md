# Contributing

This repository (`notiky/.github`) contains the **GitHub organization profile** and public-facing developer marketing materials for [Notiky](https://notiky.com). It does not contain product source code.

## What belongs here

- Org profile README (`profile/README.md`)
- Public assets (logos, banners, integration icons)
- Agent-readable docs (`AGENTS.md`, `llms.txt`)
- Contributing and security policy for this repo

## What does not belong here

- Product source code → private `notiky/notiky` repo (team access only)
- CLI bugs and feature requests → [notiky-cli](https://github.com/notiky/notiky-cli/issues)
- Internal engineering docs → not published

## Reporting issues

| Type | Where |
|------|-------|
| Product bugs or feature requests | [notiky.com](https://notiky.com) support |
| CLI installation or daemon issues | [notiky-cli issues](https://github.com/notiky/notiky-cli/issues) |
| Org README / public docs corrections | Issues or PRs on this repo |
| Security vulnerabilities | [SECURITY.md](./SECURITY.md) |

## Pull requests

PRs welcome for:

- README copy improvements (keep factual, no unbacked marketing claims)
- Asset updates (logos, banners, screenshots)
- llms.txt / AGENTS.md accuracy updates

Please keep messaging aligned with:

- Primary: *The product engineering brain*
- Secondary: *The AI workspace where products are built — from first idea to scale*

Do not include investor materials, internal roadmaps, or production credentials.

## Code of conduct

Be respectful. This is a public-facing repo representing the Notiky project.
